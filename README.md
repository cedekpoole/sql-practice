# Northwind SQL Practice

A PostgreSQL practice project using the Northwind dataset to build SQL querying, relational database understanding, data analysis and data quality checking skills.

## Project Structure

- `data/` contains the Northwind SQL dump.
- `exercises/` will contain SQL practice files.
- `notes/` will contain learning notes, query patterns, and checks.

## Focus Areas

- Reading relational schemas
- Understanding table grain
- Writing clear PostgreSQL queries
- Validating row counts and results
- Building reusable analysis habits
